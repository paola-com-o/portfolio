Funções

Uma função tem como objetivo ajudar o ser humano a realizar contas e equações onde existe uma variável dependente. Podemos usar como exemplo um taxi, o valor da corrida depende da quilometragem feita, ou seja, a váriavel independente é a quilometragem, e a váriavel dependente é o preço da corrida. Com uma equação para isso, conseguiriamos descobrir quanto o passageiro pagaria. Para escrever uma equação onde existe uma variavel independente e outra dependente geralmente é usado "f ( x )", lê-se "f de x".

É possivel que uma função seja representada por conjuntos, mas isso só pode acontecer se nao sobrarem itens e se cada item do conjunto A representar apenas um item específico do intem B.



Tipos de funções

Existem varios tipos, cada uma é usada para uma ocasião diferente. Os tipos mais usados sao:

  • Função afim: f ( x ) = 209x + 3
  • Função racional: f ( x ) = 6/x + 3
  • Função quadrática: f ( x ) = x2 + 5x - 2
  • Função modular: f ( x ) = | 12x - 8 |
  • Função exponencial: f ( x ) = 9x
  • Função logaritmica: f ( x ) = log2 (x + 4)
  • Função trogonométrica: f ( x ) = 4 sen( x)

Um exemplo de exercicio envolvendo função afim seria:

Um representante comercial recebe, mensalmente, um salário composto de duas partes: uma parte fixa no valor de R$ 3.400,00 e uma parte variável, que corresponde a uma comissão de 6% sobre o total de vendas que ele faz durante o mês. Qual a função que descreve essa relação entre as vendas realizadas e o salário recebido no final do mês? (fonte: slide da aula, autoria de Valeria Lessa)
